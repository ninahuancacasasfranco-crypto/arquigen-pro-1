# 🏛️ ArquiGen Pro: Generador Arquitectónico Paramétrico

**ArquiGen Pro** es una herramienta web interactiva diseñada para la automatización de distribuciones arquitectónicas en lotes estándar de **6m x 15m (90m²)**. El sistema genera propuestas de planta baja cumpliendo con criterios espaciales y técnicos.

## 🚀 Características Principales
- **Escala Técnica 1:50:** Visualización precisa con grosores de muro reales (0.15m).
- **Distribución Paramétrica:** 3 modelos de vivienda (Lineal, Patio Central y Loft).
- **Cálculo de Áreas en Tiempo Real:** Control de área techada y área libre según normativas (RNE).
- **Simbología Arquitectónica:** Incluye niveles de piso terminado (N.P.T.), mobiliario básico y cotas automáticas.
- **Interfaz Dark Mode:** Optimizada para reducir la fatiga visual, inspirada en software CAD profesional.

## 🛠️ Tecnologías Utilizadas
- **HTML5 / SVG:** Para la renderización de gráficos vectoriales precisos.
- **CSS3:** Diseño responsivo y estética de software técnico.
- **JavaScript (Vanilla):** Lógica paramétrica para el cálculo de áreas y generación de geometría.

## 📐 Especificaciones Técnicas
El motor de dibujo utiliza un sistema de coordenadas métricas donde `1 unidad = 1 metro`. 
- **Muros:** 0.15m de espesor.
- **Cotas:** Líneas de extensión y cifras decimales de precisión.
- **Escala Gráfica:** Barra de referencia de 2 metros incluida en el lienzo.

## 📋 Cómo usarlo
1. Clona el repositorio o abre el https://ninahuancacasasfranco-crypto.github.io/arquigen-pro-1/.
2. Selecciona una de las 3 propuestas en el panel izquierdo.
3. Observa cómo el gráfico y las estadísticas de área se actualizan instantáneamente.
